# Google Services (Firebase)

The release build can run with or without `google-services.json`.

- **Without it:** The app builds; Firebase features (e.g. Auth) will not work until you add the file.
- **With it:** Download `google-services.json` from [Firebase Console](https://console.firebase.google.com/) (Project settings > Your apps > Android app with package `edu.byu.safe_scales`), then place it in this directory (`android/app/google-services.json`).
